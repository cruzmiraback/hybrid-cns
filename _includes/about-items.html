Mobile Web Application Development

ICEsoft Technologies Inc. is the proud sponsor of BridgeIt, an open source mobile technology that simplifies mobile web application development. BridgeIt is the easiest way for you to add powerful native capabilities to your HTML5 applications. ICEsoft is also the proud sponsors of other open source mobile projects: ICEmobile, ICEfaces, and ICEpdf. We are pleased to welcome BridgeIt to our open source mobile portfolio.
